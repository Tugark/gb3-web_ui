Bildschirmgrösse zu klein

In der aktuellen Prototypphase gibt es noch keine optimierte Version für mobile Geräte oder kleine Bildschirmauflösungen.

Bitte verwenden Sie ein Gerät oder ein Browserfenster, dass mindestens eine Breite von 960px aufweist (z.B. Tablets im Landscape-Modus oder normale Browserfenster).

Sobald eine mobile Version zur Verfügung steht, können Sie den GIS-Browser auf allen Geräten mit einer angepassten Bedienung nahtlos nutzen.
